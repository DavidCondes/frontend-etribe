Acerca del proyecto

Descripción

El presente proyecto es un pequeño ejercicio que implementa las operaciones CRUD sobre una base de datos hipotética de alumnos y las posibles materias que pueden cursar. Es un proyecto simple que únicamente tiene el objetivo de practicar y demostrar habilidades requeridas para el puesto de programador JR.

Backend

La API REST fue desarrollada con el lenguaje de programación Java y el framework web Spring Boot. Asimismo, el gestor de base de datos utilizado fue MySQL.

Frontend

Para el desarrollo de la interfaz de usuario se usó principalmente el framework Angular, el cual requiere el manejo de HTML, CSS y JavaScript. De igual manera, se usó el framework Bootstrap de HTML para lograr una interfaz agradable en un corto tiempo.
